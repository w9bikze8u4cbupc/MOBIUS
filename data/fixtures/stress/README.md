# Stress Test Fixtures

This directory contains **synthetic, deterministic PDF fixtures** for stress testing MOBIUS v1.

## Overview

All PDFs in this directory are:
- ✅ **Synthetic** - Generated programmatically, not real rulebooks
- ✅ **Deterministic** - Can be regenerated identically
- ✅ **Repo-safe** - No copyrighted content, small file sizes (<3KB each)
- ✅ **Clearly labeled** - Every page marked "SYNTHETIC TEST FIXTURE"

## Design Notes

### Why Synthetic?

1. **Legal**: No copyright concerns, freely redistributable
2. **Deterministic**: Can be regenerated identically for verification
3. **Small**: Total size <10KB, suitable for git repository
4. **Controlled**: Precisely trigger intended failure modes

### Why Not Real Rulebooks?

1. **Copyright**: Commercial rulebooks cannot be redistributed
2. **Size**: Real PDFs are typically 5-50MB
3. **Variability**: Real PDFs may change between editions
4. **Licensing**: Unclear legal status for test fixtures

### Limitations

These synthetic PDFs may not trigger all the same extraction paths as real scanned rulebooks (e.g., true OCR processing). However, they reliably trigger the governance gates we're testing:

- Low-confidence extraction → `confirm_metadata`
- Missing components → `confirm_components`
- Ambiguous content → operator confirmation required

This is sufficient to validate that the governance model works correctly under adversarial conditions.

## Security Note

These PDFs are for testing only and contain no sensitive information. They are intentionally malformed or adversarial to test system robustness.
