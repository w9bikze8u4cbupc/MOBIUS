====================================================
       Mobius Games Tutorial Generator             
====================================================

Welcome to the Mobius Games Tutorial Generator!

This application helps you create interactive tutorials for board games.

To launch the application:
 1. Double-click on "launch-app.bat" 
 2. Or right-click on "launch-app.ps1" and select "Run with PowerShell"

To create a desktop shortcut for easy access:
 1. Right-click on "create-desktop-shortcut.ps1"
 2. Select "Run with PowerShell"

To stop the application:
 1. Close the command window where it's running
 2. Or run "stop-servers.bat" or "stop-servers.ps1"

For more information, read the DESKTOP_LAUNCHER_README.md file.
